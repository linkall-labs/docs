# Output Events of Auth0 Source

The Auth0 Source supports various types of [Auth0 Webhook events](https://auth0.com/docs/logs/references/webhooks).

The Auth0 Source generates [CloudEvents](https://docs.vanus.ai/reference/cloudevents) for each type of Auth0 event.
