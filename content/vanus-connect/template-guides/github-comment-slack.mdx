# GitHub issue comment event triggers a Slack message

GitHub is the largest web-based platform that provides developers with a place to host and manage their software projects.

You can receive notifications when specific events occur in a repository by adding a GitHub Source on Vanus Connect.

This guide will help you get a Slack notification whenever someone leaves a comment in your repository.
